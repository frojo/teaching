Randomness

Unity has a pretty nice Random class that lets you randomize certain things.

For example…

Random.Range()

Random.Range(float min, float max) lets you get a random number between a specified range.

Let’s say you wanted to get a random number between 0 and 13:

float randomNumber = Random.Range(0f, 13f);

Note that this will give you a float (a decimal). If you wanted an integer you could do:

int randomInt = Random.Range(0, 14);
// note that it's 14 instead of 13

You can use this to choose a random string from an array of strings

string[] shopping_types = {"Wandering Shopper", "carrot", "pepper"};

string randomIngredient = ingredients[Random.Range(0, ingredients.Length))];

Coroutines

Coroutines are special functions that execute code over time.

They’re used to code anything that is time dependent,

  • timers
  • dialog
  • turn order in strategy games
  • Enemy Behavior

Coroutines look like other functions, except they have an IEnumerator type.

Here’s how you would write a coroutine that prints something after 5 seconds.

using UnityEngine;
using System.Collections;

public class ExampleClass : MonoBehaviour
{

    void Start()
    {
        // - After 0 seconds, prints "Starting 0.0 seconds"
        
        print("Starting " + Time.time + " seconds");
        // Start function WaitAndPrint as a coroutine.
        StartCoroutine(WaitAndPrint(););
    }

    private IEnumerator WaitAndPrint()
    {
        yield return new WaitForSeconds(5);

				// - After 5 seconds, prints "Coroutine ended: 2.0 seconds"

        print("Coroutine ended: " + Time.time + " seconds");
    }
}
