Simple Animation | DUE 9/13

Let's get started using some basic scripts in Unity, by making some sprites and moving them around.

▶️ IN UNITY: Create a "poetic landscape" based on a feeling or dream. Position the camera and populate a scene with sprites that move.
  • Have at least 5-10 sprites, including scene objects, a background, and the elements that move. You can use physics to make things move, but you must write at least 2 scripts that control something in the scene.
  • No input or gameplay allowed, just make a somewhat complex static 3D scene.
  • build out to WebGL, upload to itch.io, and submit it on Canvas
