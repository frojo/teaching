Autobiographical Game | DUE 12/11

▶️ IN UNITY: Use what we learned this semester to make your own autobiographical game!

Choose a memory, moment, or aspect of your real life / daily routine / identity, and express it as a short interactive experience. It can be realistic, abstract, fantastical, or metaphorical. Take inspiration from the examples in the Ludography, and the two talks by Catt Small and Jenny Jiao Hsia.

Requirements:

  • keep it small (you have 3 weeks), keep it short
  • must have 1 interaction / mechanic / verb other than basic movement (e.g. jump, pick-up, sing, dance, hide, etc
  • You can use 3D models, 2D sprites, or both
  • must maintain a Git repo on GitHub with frequent commits

Upload your working build on Itch.io, and submit it on Canvas, along with a link to your git repo.
