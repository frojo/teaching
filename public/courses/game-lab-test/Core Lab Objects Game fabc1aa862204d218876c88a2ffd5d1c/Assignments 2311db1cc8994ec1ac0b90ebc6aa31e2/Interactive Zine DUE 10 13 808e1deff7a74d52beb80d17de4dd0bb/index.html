▶️ IN UNITY: Make your own interactive zine!

Zines have a long history as creative works of personal expression

You can make a zine about anything! What's something you'd want to tell your peers about?

Check out How to Make a Zine or Zine Machine! for some ideas

Here’s *a ton* of examples (just check out a few).

(Keep in mind all of these zines are non-interactive. Yours should be interactive!)

  • Use sprites (with found or original artwork), text, and animations. You can also use 3D models if you'd like, and anything else we've learned about so far.
  • It should feature at least 4 pages of information
  • build out to WebGL, upload to itch.io, and submit it on Canvas

Script for changing camera with button

using System.Collections;
using System.Collections.Generic;
using UnityEngine;

public class ChangeCameraButton : MonoBehaviour
{

    public Camera nextCamera;
  

    public void ChangeCamera() {
        
        // disable the main camera
        Camera.main.enabled = false;
        
        // enable the next camera
        nextCamera.enabled = true;

    }
}
