Week 14 (Friday) - Character Controller

Why Gold Player?

There other options for drop-in first person player controller like:

  • First person drifter
  • Unity’s official FirstPerson controller

We’re looking at Gold Player because it’s the most updated package that is still open source

Basics

GoldPlayerController.cs

You can open the script!!

The GoldPlayer code can seem pretty intimidating! Especially because it has a lot of boilerplate that looks like this:

[RequireComponent(typeof(CharacterController))]
    [DisallowMultipleComponent]
    [AddComponentMenu("Gold Player/Gold Player Controller", 0)]
    [HelpURL("https://github.com/Hertzole/gold-player")]
    [SelectionBase]
    public class GoldPlayerController : MonoBehaviour
    {
        [SerializeField]
        [FormerlySerializedAs("m_Camera")]
        [FormerlySerializedAs("camera")]
        private PlayerCamera cam = new PlayerCamera();
        [SerializeField]
        [FormerlySerializedAs("m_Movement")]
        private PlayerMovement movement = new PlayerMovement();
        [SerializeField]
        [FormerlySerializedAs("m_HeadBob")]
        private PlayerBob headBob = new PlayerBob();
        [SerializeField]
        [FormerlySerializedAs("m_Audio")]
        [FormerlySerializedAs("audio")]

These are Attributes that make this script a lot nicer to use in your project. There’s a lot of them! Just go one line at a time. For example:

[RequireComponent(typeof(CharacterController))]

Makes it so you need to have the a CharacterController component

More documentation: here

Try removing the CharacterController component and see what happens!

You can go through all these and use it as inspiration for you’re developing your own assets/packages!

Modules

The GoldPlayerController is split into different modules. For example the Camera:

 				[SerializeField]
        [FormerlySerializedAs("m_Camera")]
        [FormerlySerializedAs("camera")]
        private PlayerCamera cam = new PlayerCamera();

But what’s PlayerCamera? And where does all this stuff come from?

Looking through the folders we find… PlayerCamera.cs!
