Lerping aka Linear interpolation

Linear interpolation, or "Lerping", in Unity can be accomplished with the Mathf.Lerp() function. This function takes three parameters: the start value, the end value, and the interpolation value (which should be between 0 and 1). Here's an example:

float startValue = 0f;
float endValue = 10f;
float interpolationValue = 0.5f;

float result = Mathf.Lerp(startValue, endValue, interpolationValue);

// result will be 5, because it's halfway between startValue and endValue

This function can be used in a variety of situations. For instance, you can use it to smoothly transition a game object from one position to another.

Tweening

Tweening can often be used to make your games, feel more polished or professional.

Tweening in Unity can be accomplished using the Animation system, or by using external libraries such as DOTween. Here's an example of how to perform a simple tween using DOTween:

using DG.Tweening;

public class Example : MonoBehaviour
{
    private void Start()
    {
        // Move the game object to position (10, 10, 10) over 3 seconds
        transform.DOMove(new Vector3(10, 10, 10), 3);
    }
}

In this example, the game object to which this script is attached will move to the position (10, 10, 10) over a period of 3 seconds. Tweening like this can be used to create smooth, complex animations with minimal coding.

DOTween is the standard way of doing tweening in unity.
