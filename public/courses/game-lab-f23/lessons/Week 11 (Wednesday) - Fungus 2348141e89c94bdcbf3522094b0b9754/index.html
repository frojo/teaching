Fungus

“Fungus is a tool which makes it easy to add colorful characters and craft gripping story lines in your game!”

Fungus is a great tool for creating visual novels in Unity.

Installing Fungus

Download this Unity package:

Fungus_v3.13.8.unitypackage

You can also get it from here: https://github.com/snozbot/fungus/releases/tag/v.3.13.8

And then import it into a new 2D Unity project

Exercise: Create a scene of two characters having a conversation. Include:

  • at least 4 blocks
  • Using the Menu block to have an interactive choice
  • At least one portrait for each character
  • Use a Stage to make the portraits show in some way that isn’t the default
