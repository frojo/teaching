On Blockouts

A blockout (also blockmesh or graybox) is a 3D rough draft level built with simple 3D shapes, but without any details or polished art assets.

It is "cheap" to delete or rebuild some rough blockout geometry, but throwing away finalized

ProBuilder

A useful tool for blocking out levels and environments.

How to install ProBuilder

Install it from the Package Manager

  1. Open the package manager: Window → Package Manager

  2. Make sure you set Packages to Unity Registry

  3. in the search bar search for ProBuilder

How to use ProBuilder

Open the ProBuilder window: Tools → ProBuilder → Window

I like to drag it to the scene view like this

Creating shapes

Click New Shape to create a shape

There a few different shape primitives to choose from.

You can also click New Poly Shape to create a custom shape.

Editing shapes

Like with Blender, you can be in four different selection modes: object, vertex, edge, and face

These different modes will let you select and move different parts of your shapes.

Different modes also have their own editing affordances.

For example, in edge mode you can select the cut tool. This similar to making cuts in Blender.

You can also collapse vertices and extrude faces etc.
