UI Masks

A Mask is not a visible UI element but rather a way to modify the appearance of child elements. The mask restricts (ie, "masks") the child elements to the shape of the parent.

How to create a mask

To create a Mask, first create an Image UI element (Assets > Create > UI > Image).

And then add a Mask component. It should look like this:

Now you can add another Image child (don’t make it a Mask)

And that’s really it!

Good textboxes/frames

This is how you can make texboxes with borders scale well in Unity.

First, open up your sprite in the Sprite Editor

Then, move the green edges so that the corners of the sprite are framed.

Then, when you create the gameobject with an Image component, change the Image Type to Sliced

Now it should scale properly!

Making a scroll area

Unity comes with an easy way to create a scrollable area.

Create a new Scroll View by right-clicking your Canvas and selecting UI → Scroll View

This will create a pretty complex scrollable area that comes with scrollbars

The important thing is that whatever you want to put in this area, you add it as a child to the Content object
