Controlling particle systems with scripts

Dog game assets to follow along:

dog-game-assets.unitypackage

To control particle systems using scripts in Unity, you can use the ParticleSystem component and various properties and methods available. Here are the basic steps:

  1. Attach a ParticleSystem component to the GameObject you want to control.
  2. Access the ParticleSystem component in your script using GetComponent or by assigning it in the inspector
  3. Use Emit() how to emit particles when you want to

Here his some example code:

public class ExampleCode : MonoBehaviour
{
    public ParticleSystem particleSystem;

    private void Start()
    {
        // Emit particles
        particleSystem.Emit(10);
    }
}

This code will emit ten particles when you start your game.

You can also emit particles per frame, for example when your character is moving. Here’s an example of that.

using System.Collections;
using System.Collections.Generic;
using UnityEngine;

public class Player : MonoBehaviour
{
    public float speed;

    public ParticleSystem particles;

    // Start is called before the first frame update
    void Start()
    {

        
    }

    // Update is called once per frame
    void Update()
    {

        if (Input.GetKey(KeyCode.W) || Input.GetKey(KeyCode.UpArrow)) {
            transform.position += new Vector3(0, speed, 0) * Time.deltaTime;
            particles.Emit(1);
        }

        if (Input.GetKey(KeyCode.S) || Input.GetKey(KeyCode.DownArrow)) {
            transform.position += new Vector3(0, -speed, 0) * Time.deltaTime;
            particles.Emit(1);
        }

        if (Input.GetKey(KeyCode.A) || Input.GetKey(KeyCode.LeftArrow)) {
            transform.position += new Vector3(-speed, 0, 0) * Time.deltaTime;
            particles.Emit(1);
        }

        if (Input.GetKey(KeyCode.D) || Input.GetKey(KeyCode.RightArrow)) {
            transform.position += new Vector3(speed, 0, 0) * Time.deltaTime;
            particles.Emit(1);
        }
        
    }
}
